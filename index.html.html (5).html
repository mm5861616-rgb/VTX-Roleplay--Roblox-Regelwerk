VTX Roleplay — Regelwerk

Hier findest du die offiziellen Regeln für den VTX Roleplay Server. Wähle eine Kategorie, um die Regeln einzusehen.

Allgemeine Regeln

Anzeigen ▾
  • Allg. 1.1 Sobald ein User mit seinem Account dem Roleplay-Server joint, darf nur er mit diesem Account auf dem Server spielen. Permanenter Bann
  • Allg. 1.2 Ingame-Eigentum darf nicht für Echtgeld verkauft werden. Permanenter Bann
  • Allg. 1.3 Das Besitzen von über 3 Accounts auf dem Roleplay-Server ist verboten. Main-Account: 60 Tage Bann; alle anderen Accounts: permanente Sperre vom Server. 60 Tage Bann / Permanent
  • Allg. 1.4 Eigentum wie Waffen oder Geld darf nicht zwischen Main- und Twink-Accounts getauscht werden. Permanenter Bann für beide Accounts. Permanenter Bann
  • Allg. 1.5 Um auf unserem Server spielen zu können, wird Discord benötigt. Permanente Sperre (bei Nicht-Nutzung)

Kommunikations Regeln

Anzeigen ▾
  • Komm. 1.1 Um auf unserem Roleplay-Server spielen zu wollen, wird Ingame-Voice benötigt. Permanenter Bann (Entbannung über Discord möglich)
  • Komm. 1.2 OOC-Talk ist nicht erlaubt — Real-Life-Gespräche sind untersagt. Admin Jail 15 Minuten
  • Komm. 1.3 IC / OOC-Vermischung ist untersagt — OOC-Situationen dürfen nicht IC benutzt werden. Admin Jail 30 Minuten + Warnung
  • Komm. 1.4 Das Ticketsystem darf nicht ausgenutzt werden. Admin Jail 25 Minuten + Warnung

IC Regeln

Anzeigen ▾
  • IC 1.1 VDM — Vehicle Deathmatch ist verboten. Admin Jail 20 Minuten
  • IC 1.2 CR — Car Ramming ist verboten. Admin Jail 20 Minuten
  • IC 1.3 Fail RP — unrealistisches Roleplay ist untersagt. Admin Jail 20 Minuten
  • IC 1.4 RDM — Random Deathmatch ist untersagt. Admin Jail 25 Minuten
  • IC 1.5 PG — Powergaming ist verboten. Admin Jail 30 Minuten

Leader Regeln

Anzeigen ▾
  • Leader 1.1 Ein Leiter einer Organisation muss bis zu 14 Stunden in der Woche online sein. Orgawarn
  • Leader 1.2 Ein Leiter einer staatlichen Organisation darf höchstens 2 Vize-Leiter ernennen. Orgaclose + Bann 60 Tage
  • Leader 1.3 Ein Leiter einer kriminellen Organisation darf höchstens 4 Vize-Leiter ernennen. Orgaclose + Bann 60 Tage
  • Leader 1.4 Ein Leiter darf nicht Teil eines anderen Roleplay-Servers sein. Permanenter Bann
  • Leader 1.5 Ein Leiter darf nur mit Genehmigung des Beraters ein Mitglied wegen Vertrauensverlusts kicken. Orgawarn
